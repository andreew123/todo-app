# Todo App

## Description
- A simple task handler application with Laravel 5.4 api and Angular 5 frontend.

## Install Laravel 5.4
- `clone repository`
- `rename .env.example file to .env`
- `composer install`
- `php artisan key:generate`
- `php artisan serve`

## Create database
- `create new database 'todo'`
- `php artisan migrate`
- `php artisan db:seed`

## Install Angular 5
- `npm install`
- `npm run dev`
